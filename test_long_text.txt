# 长文本脱敏测试实例

以下是一个包含多种敏感信息的长文本实例，适合用于测试脱敏模型，同时脱敏后仍能保持足够的语义信息供大模型理解和回答：

---

**客户咨询案例 - 投资理财建议**

客户姓名：张明远，身份证号码：110101198507154321，联系电话：13812345678，电子邮箱：zhangmingyuan@example.com

客户于2023年5月15日通过腾讯金融APP咨询关于投资理财的建议。客户目前就职于北京科技创新有限公司，担任产品经理职位，月薪约25,000元。客户住址位于北京市海淀区中关村大街123号，邮编100080。

客户表示目前有约50万元的闲置资金，希望进行合理的投资配置。客户的风险承受能力评估为中等，投资期限为3-5年。客户的主要投资目标是在控制风险的前提下实现资产的稳健增值。

客户家庭情况：已婚，配偶李晓华在华为技术有限公司担任高级工程师，月薪约30,000元。有一个5岁的女儿，正在上幼儿园。家庭月收入约55,000元，月支出约20,000元，包括房贷8,000元、生活费7,000元、子女教育费用3,000元和其他支出2,000元。

客户现有资产情况：
1. 银行存款：20万元（工商银行账号：6222020200012345678）
2. 股票投资：15万元（主要持有腾讯控股、阿里巴巴等科技股）
3. 基金投资：10万元（混合型基金）
4. 房产：一套位于北京朝阳区的公寓，市值约500万元，贷款余额200万元（招商银行贷款，账号：6225880200019876543）

客户咨询的具体问题：
1. 如何配置这50万元的闲置资金？
2. 是否应该增加股票投资比例？
3. 有哪些适合中等风险偏好的理财产品？
4. 如何平衡短期流动性和长期收益？
5. 是否需要考虑为孩子准备教育基金？

客户表示对数字货币和区块链技术有一定了解，但担心其高风险特性。客户还提到计划在3-5年内购买第二套房产，需要准备首付款。

客户的时间偏好：希望能够在工作日的晚上或周末进行投资决策，不希望花费太多时间管理投资组合。

客户的技术背景：计算机专业毕业，对金融科技产品接受度高，愿意使用APP进行投资管理。

客户的特殊要求：
1. 希望投资产品具有良好的流动性
2. 优先考虑有腾讯背书或合作的金融产品
3. 希望能够定期获取投资组合的表现报告
4. 对ESG（环境、社会和治理）投资理念感兴趣

客户的风险提示：客户曾于2021年投资某P2P平台损失约5万元，因此对新型投资产品较为谨慎。

客户的联系方式补充：微信号：zhangmy_invest，QQ号：123456789，办公电话：010-88889999

---

这个文本包含了以下类型的敏感信息：
1. 个人身份信息（姓名、身份证号）
2. 联系方式（电话、邮箱、微信、QQ、办公电话）
3. 金融信息（银行账号、贷款账号、资产详情）
4. 职业信息（公司名称、职位、薪资）
5. 地址信息（家庭住址、办公地址）
6. 家庭成员信息（配偶姓名、公司、职位、薪资）

脱敏后的文本应该保留足够的语义信息，使大模型能够：
1. 理解客户的基本情况和需求
2. 分析客户的财务状况和风险偏好
3. 提供合适的投资建议
4. 推荐适合的理财产品
5. 解答客户的具体问题
